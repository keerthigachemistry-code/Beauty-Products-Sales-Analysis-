# Beauty Products Sales Analysis 💄

Self-initiated data analysis project analyzing sales of 9 beauty products across 2025–2026.

## Tools Used
- Microsoft Excel
- SQL 
- Power bi

## Key Results
- Total Sales grew 59.8% ($10,840 → $17,320)
- Net Profit grew 700% ($350 → $2,800)
- Losses reduced by 42.2%
